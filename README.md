# 🎨 我的私人 Loon 图标库

欢迎使用我的私人图标库！这里专门存放我个人喜好的策略组图标。

## 📥 如何导入
在 Loon 中长按策略或插件 **图标** -> **右上角＋号** -> **从 URL 导入**，粘贴下方链接：

`https://raw.githubusercontent.com/Shamana-cn/My-Loon-Icons/main/my_icons.json`
